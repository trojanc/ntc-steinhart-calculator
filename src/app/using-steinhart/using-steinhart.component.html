Using Steinhart-hart coefficients

Now that you have the coeffiecients, let's revisit what we have learned:

  1. Micro controllers don't directly read resistance, but rather an analog (voltage) level.
  2. We need to configure a circuit that allows us to use the NTC in a configuration that is most effective for the temperature range we would like to measure.
  3. We need to work out the coefficients so that we can acurately apply math to work out the temperature for a given resistance.
  4. We need to determine the NTC resistance at the given analog voltage reading.
  5. Once we have the resistance of the NTC, we can use the coefficients to calculate the temperature.
